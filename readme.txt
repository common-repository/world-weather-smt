== Description ==

World Weather est une extension pour wordpress qui vous permet de collecter des données d'un fournisseur meteo (gratuit ou payant) pour les afficher ensuite dans vos contenus : pages et articles, mais aussi sous forme de widget.

== Installation ==

1. Uploadez les fichiers de l'extension dans le répertoire '/wp-content/plugins/world-weather-smt', ou installez le à travers le répertoire d'extension directement
2. Activez l'extension
3. Utilisez l'écran dans Réglages > World Weather pour configurer le plugin
4. Vous devez configurer au moins un fournisseur de données et une localisation par défaut

==  Third party services ==

Fournisseurs météo implémentés : OpenWeather, Accuweather (nous travaillons à en ajouter d'autres.)

Pour la géo-localisation, nous utilisons GEONAMES (geonames.org) et son service geocode pour obtenir les coordonnées geo des adresses fournies en texte.  https://geonames.org - Service : GEOCODE
